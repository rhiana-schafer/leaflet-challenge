# leaflet-challenge
This project contains code for a visualization of worldwide earthquakes over the last 7 days, using data from the USGS.

## Methods Used  
* D3
* Javascript
* Plotly
* Leaflet
* HTML
* API calls

## Data Source
No API call needed.
https://earthquake.usgs.gov/earthquakes/feed/v1.0/summary/all_day.geojson
